Tennis de Table

Le Tennis de table aussi appelé "Ping-Pong", est un jeux de raquettes pouvant se jouer à 2 ou à 4. Pour gagner un match de tennis de table il faut marquer 3 settes de 11 points. Le tennis de table à été inventé en Angleterre au XXème siècle. C'est devenue une sport olympique en 1988. Le tennis de table à long temps été dominé par les Européens. Mais à partir de sont arrivé aux Jeux Olympique, les Chinois ont commencé à dominer le Tennis de table mondiale.
